Projetos

API de Gerenciamento de Usuários, Produtos e Carrinhos (FastAPI)

Desenvolvi uma API RESTful com FastAPI para simular operações básicas de um sistema de e-commerce. A aplicação permite o cadastro de usuários e produtos, criação de carrinhos de compras, adição de itens ao carrinho e visualização dos itens selecionados.

  • Criação de endpoints REST com FastAPI para operações de cadastro e manipulação de dados
  • Organização modular do código em classes separadas para usuário, produto e carrinho
  • Tratamento de erros e validação de existência de entidades com HTTPException
  • Geração de identificadores automáticos e armazenamento dos dados em memória
